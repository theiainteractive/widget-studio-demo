# Overview
This repository contains the demo project showcasing the capabilities of the [Widget Studio](https://www.unrealengine.com/marketplace/en-US/product/widget-studio).

# Requirements

| Name | Requirement |
| -- | -- |
| OS | Windows |
| Unreal Engine Version | 4.26 |
| Plugins | [Widget Studio](https://www.unrealengine.com/marketplace/en-US/product/widget-studio) |

# Instructions
Follow these instructions to get up-and-running with the Widget Studio Demo

## 1. Download Repository
You can download the repository via the `Download ZIP` button under `Code`. Once downloaded, unzip the file.

![image](https://user-images.githubusercontent.com/21698404/134955021-5aca0e41-4acc-4467-8987-c0d3745367c3.png)

Alternatively, you can clone the repository utilizing Git Bash, Github Desktop or an alternative software. 

## 2. Open the Widget Studio Demo project
Once unzipped, you can open the Unreal project at `widget-studio-demo/src/WidgetStudioDemo.uproject`

## 3. Viewing the demo files
In the Content Browser inside of the Unreal Editor, there will be multiple folders. The primary one to view will be labeled `WidgetStudioDemo`. All others in the root are resources used to make up the demo.

# Additional Information
- The level for the Main Menu is `WidgetStudioDemo_MainMenu` and it can be found at `Content/WidgetStudioDemo/Maps`
- The level for the content is `WidgetStudioDemo_ShooterMap` and it can be found at `Content/WidgetStudioDemo/Maps`
- Check out the level blueprints to see how things are hooked up
- Play In Editor to play the demo
